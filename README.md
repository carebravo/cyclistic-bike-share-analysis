# Cyclistic Bike-Share Analysis

## Business Objective

Analyze how annual members and casual riders use Cyclistic bikes differently and provide recommendations to increase annual memberships.

## Tools Used

- SQL
- Google Sheets
- Tableau
- GitHub

## Key Findings

1. Members generate more trips.
2. Casual riders take longer trips.
3. Casual riders prefer weekends.
4. Members exhibit commuting behavior.
5. Seasonal trends affect casual riders more.

## Recommendations

- Promote membership value.
- Target weekend riders.
- Focus marketing during peak seasons.
- Offer membership incentives.
